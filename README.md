# khaledashraf.me

My personal corner of the internet. I'm Khaled, a product designer bridging design and code to make beautiful, functional things.

This portfolio is built and designed by me, from scratch. No templates.

## What's here

## Stack

- Next.js 16
- Tailwind CSS v4
- Framer Motion
- shadcn/ui
- MDX for blog posts
